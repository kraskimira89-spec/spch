# Проект контент-анализа

## Актуальный процесс

Проект сейчас работает вокруг пересчёта **полного корпуса** по **7 укрупнённым смысловым кодам** и сборки итогового отчёта в `output/`.

### Основные скрипты

- `scripts/rebuild_output.py`  
  Полная пересборка папки `output/` с очисткой старых артефактов.

- `scripts/recount_semantic_codes.py`  
  Пересчёт полного корпуса по 7 укрупнённым смысловым направлениям.

- `scripts/build_semantic_tone_report.py`  
  Excel-отчёт: смысловые категории × тональность, сводные таблицы и диаграммы.

- `scripts/build_tone_word_report.py`  
  Отдельный Word-отчёт по тональности в оформлении основного аналитического отчёта.

- `scripts/rebuild_itog_report.py`  
  Сборка итогового Word-отчёта на основе актуального пересчёта.

### Вспомогательные скрипты

- `scripts/archive_legacy_files.py`  
  Перенос лишних и устаревших файлов проекта в `Архив/`.

- `scripts/archive_old_scripts.py`  
  Перенос неактуальных старых скриптов в `Архив/Старые_скрипты/`.

### Важно

- `scripts/build_by_tz.py` пока **не архивирован**, потому что от него зависят функции:
  - чтение `ТЗ контент.docx`;
  - разбиение корпуса на атомарные высказывания;
  - базовая подготовка текста для пересчёта.

## Основная команда

```bash
python scripts/rebuild_output.py
```

### Состав `output/`

- `output/01_Отчет/01_Итоговый_отчет.docx`  
  Основной аналитический Word-отчёт по полному корпусу, построенный на 7 укрупнённых смысловых кодах.

- `output/01_Отчет/02_Отчет_по_тональности.docx`  
  Отдельный аналитический Word-отчёт по тональности: 4 тональные категории, таблицы, диаграммы и кросс-анализ со смысловыми кодами.

- `output/02_Пересчет_смысловых_кодов/01_Матрица_смысловых_кодов.xlsx`  
  Полная кодировочная матрица по 7 укрупнённым смысловым кодам для атомарных высказываний.

- `output/02_Пересчет_смысловых_кодов/02_Пояснительная_записка_по_смысловым_кодам.docx`  
  Пояснение логики укрупнения кодов, состава категорий и итогов пересчёта.

- `output/02_Пересчет_смысловых_кодов/03_Итоги_смысловых_кодов.txt`  
  Краткая текстовая сводка итогов пересчёта по смысловым кодам.

- `output/02_Пересчет_смысловых_кодов/04_Отчет_по_тональности.xlsx`  
  Excel-отчёт: смысловые категории × тональность, сводные таблицы и диаграммы.

- `output/03_Исходные_данные/01_ТЗ_контент.docx`  
  Основной исходный документ, из которого извлекается корпус ответов.

- `output/03_Исходные_данные/02_ТЗ_контент.pdf`  
  PDF-копия исходного документа.

- `output/03_Исходные_данные/03_Методика_контент_анализа.md`  
  Методическое описание подхода, на котором основаны пересчёт и интерпретация результатов.

- `output/README.txt`  
  Краткая справка по составу папки `output/` для быстрого просмотра вне репозитория.

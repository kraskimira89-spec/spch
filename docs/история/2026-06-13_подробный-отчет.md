# 2026-06-13 — Подробный цветной отчёт

## Что сделано

- Скрипт `scripts/build_detailed_report.py`: Word-отчёт с цветными таблицами, инфо-блоками, встроенными диаграммами.
- Подключён в `scripts/build_by_tz.py` (перед упаковкой ZIP).
- Дополнительные диаграммы в `04_Диаграммы/`: `отчет_ранжирование_тем.png`, `отчет_тональность.png`, `отчет_сводка.png`.

## Файл отчёта

`output/Контент-анализ_по_ТЗ/05_Отчет/Подробный_отчет_контент-анализ.docx`

## Секции отчёта

1. Титул и резюме для руководителя
2. Введение
3. Методика (10+3 коды, таблица параметров)
4. Количественные результаты (ранжирование, тональность, рисунки)
5. Качественный анализ (цитаты по топ-5 тем, паттерны)
6. Практические рекомендации
7. Проверка достоверности данных

## Запуск

```bash
python scripts/build_detailed_report.py
```

или полная пересборка:


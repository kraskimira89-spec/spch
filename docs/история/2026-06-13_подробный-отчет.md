## Пересчёт по смысловым кодам

- Скрипт `scripts/recount_semantic_codes.py`: пересчёт полного корпуса по 7 укрупнённым смысловым направлениям без регионального деления.
- Результат: `output/02_Пересчет_смысловых_кодов/`.
- Файлы:
  - `01_Матрица_смысловых_кодов.xlsx` — полная кодировочная матрица по 7 укрупнённым смысловым кодам для атомарных высказываний
  - `02_Пояснительная_записка_по_смысловым_кодам.docx` — пояснение логики укрупнения кодов, состава категорий и итогов пересчёта
  - `03_Итоги_смысловых_кодов.txt` — краткая текстовая сводка итогов пересчёта по смысловым кодам
  - `04_Отчет_по_тональности.xlsx` — Excel-отчёт: смысловые категории × тональность, сводные таблицы и диаграммы
- По пересчёту тема «Экономическая модель и тарифная политика» вышла на 1 место: `146` из `609` атомарных высказываний (`24.0%`).
